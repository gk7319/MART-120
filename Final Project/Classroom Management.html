• Class Dojo

    • Class Dojo is a software where a teacher can track students who are showing classroom values.

    It is also a great tool for communication between the teacher and the caretaker.

    Point System

    Each time a kid does something that represents the classroom values, the teacher can give them a point.

    Individual points add up to create a whole classroom score.

    As the teacher, you can choose what reward to give to the students that reach a certain point value.

    Communication

    As a closed communication system, Class Dojo is a safe way to communicate between care giver and teacher.
